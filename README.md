# 🐶🐱 멍냥케어 (MeongNyang Care)

AI 이미지 분석을 활용하여  
**반려견·반려묘의 피부 질환을 예측하고 관리 가이드를 제공하는  
AI 기반 반려동물 케어 서비스**입니다.

본 프로젝트는 **대용량 실제 의료 이미지 데이터(AI Hub)**를 기반으로  
단순 모델 실험이 아닌  
**AI 모델 학습 → 추론 → 서비스 연동까지 전 과정을 구현**한  
졸업 프로젝트입니다.

---

## 📌 Project Overview
- **Type**: 졸업 프로젝트 (AI 기반 서비스)
- **Domain**: Computer Vision / Medical Image Analysis
- **Period**: 2023.03 ~ 2023.06
- **Team**: 4HIS 팀
- **My Role**: **AI 모델 학습 및 데이터 처리 (반려견 데이터 전담)**

---

## 🎯 Problem Definition

반려동물 피부 질환은 초기 증상이 유사하여  
보호자가 질환의 종류나 심각도를 판단하기 어렵고,  
병원 방문 시기를 놓치는 경우가 많습니다.

### 해결하고자 한 문제
- 피부 사진 한 장으로 **질환 종류를 예측**할 수 없을까?
- 질환의 **위치와 진행 상태**를 함께 보여줄 수 없을까?
- AI 분석 결과를 **실제 서비스 기능으로 연결**할 수 없을까?

---

## 📊 Dataset

- **출처**: AI Hub (한국지능정보사회진흥원)
- **총 데이터 용량**: 약 **382GB**
- **총 이미지 수**: 약 **50만 장**
  - 반려견: 약 446,400장
  - 반려묘: 약 53,600장

### 데이터 특징
- 실제 촬영 환경 이미지 (조명·각도·품질 편차 큼)
- JSON 기반 라벨링
  - 질환 종류
  - Bounding Box
  - Segmentation Mask
- 수의사 진단 기준 데이터

### 데이터 처리
- 대용량 데이터 구조 분석
- 학습용 이미지 정제 및 분리
- 학습/검증 데이터셋 구성
- 클래스 불균형 문제 인식 및 고려

---

## 🧠 AI Model Architecture

본 프로젝트는 **단일 모델이 아닌 파이프라인 구조**로 설계되었습니다.

### 1️⃣ 질환 분류 모델 (Classification)
- **CNN 기반 이미지 분류 모델**
- Architecture: **InceptionResNetV2**
- 입력 이미지로부터
  → 반려동물 피부 질환 종류 예측

### 2️⃣ 병변 영역 분할 모델 (Segmentation)
- **U-Net 기반 모델**
- 피부 이미지에서
  → 병변 영역을 픽셀 단위로 분할

### 3️⃣ 결과 통합
- 질환 종류
- 병변 위치
- 진행 단계 및 위험도 판단

> 단순 정확도보다  
> **“AI가 무엇을 근거로 판단했는지”를 보여주는 구조**를 중요하게 설계했습니다.

## 🔥 Training & Challenges

### 주요 어려움
- 수십만 장 규모 데이터로 인한 학습 자원 부담
- 클래스 간 데이터 편차로 인한 성능 불균형

  등 지표의 중요성 이해
- **모델 선택은 문제 정의에 따라 달라져야 한다는 점을 학습**

---

## 🧩 Service Integration

본 프로젝트는 **모델 학습에서 끝나지 않고**,  
실제 모바일 서비스 기능으로 연동하는 것을 목표로 했습니다.

### 서비스 흐름
1. 사용자가 반려동물 피부 사진 촬영
2. 서버를 통해 AI 모델 추론 요청
3. 질환 예측 결과 반환
4. 사용자에게
   - 질환 정보
   - 위험도
   - 관리 가이드 제공

> **AI → 서비스 기능으로 연결한 경험**이  
> 본 프로젝트의 가장 큰 가치입니다.

## 📈 Outcome & Learning

- 대규모 실제 데이터를 다뤄본 경험
- AI 모델을 단순 실험이 아닌 **서비스 기능으로 활용**
- 데이터 → 모델 → 서비스로 이어지는 전체 흐름 이해

> **AI를 ‘학습’하는 것에 그치지 않고  
> 실제 사용자에게 의미 있는 기능으로 연결할 수 있는 경험을 얻었습니다.**
